Notification System

Interactive toast notifications with customizable duration, types, and auto-dismiss functionality

Set to 0 for persistent notification
  • Multiple Types: Success, Error, Warning, and Info notifications
  • Auto-dismiss: Configurable duration for automatic dismissal
  • Stack Management: Display multiple notifications simultaneously
  • Smooth Animations: Slide-in and slide-out transitions
  • Icon Integration: Type-specific icons for better recognition
  • Accessibility: ARIA attributes for screen reader support
  • Responsive Design: Adapts to mobile and desktop screens
