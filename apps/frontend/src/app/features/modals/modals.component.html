Usage Examples

Code examples for common modal patterns

Simple Modal

this.modalService.open(MyComponent, {
  title: 'Modal Title',
  size: 'md',
  data: { message: 'Hello!' }
}).then(result => {
  console.log('Modal closed:', result);
});

Confirmation Dialog

const confirmed = await this.modalService.confirm({
  title: 'Confirm',
  message: 'Are you sure?',
  confirmText: 'Yes',
  cancelText: 'No'
});

if (confirmed) {
  // User confirmed
}

Alert Dialog

await this.modalService.alert(
  'This is an important message',
  'Alert'
);
